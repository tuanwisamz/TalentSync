> **Hackathon Project · Talent Bank Hackathon**

## What is TalentSync?

TalentSync is an AI-powered career intelligence platform designed to bridge the gap between job seekers and opportunities. It goes far beyond traditional keyword-matching tools — instead, it understands career narratives, extracts real impact from resumes, and generates actionable outputs that help candidates get noticed and get hired.

Built as a prototype for the **Talent Bank Hackathon**, TalentSync addresses a critical pain point: talented candidates are often overlooked not because they lack skills, but because they struggle to communicate their value effectively in a competitive hiring landscape.

## The Problem We're Solving

Modern hiring is broken in two key ways:

- **For job seekers** — Generic resumes get filtered out by ATS systems before a human ever reads them. Interview prep is scattered across multiple tools. Cold outreach is time-consuming and rarely personalised.
- **For recruiters** — Screening hundreds of inconsistent resumes wastes hours. Signal gets buried under noise.

TalentSync tackles both sides of this problem in one unified platform.

## Key Features

### Resume Analysis Engine

Parses and analyses resumes using NLP to extract structured data — roles, skills, impact statements, and hidden intent — not just keywords. Provides instant gaps, strengths, and matching insights.

### ATS Optimisation System

Evaluates resume compatibility against job descriptions using a custom scoring algorithm. Provides a keyword analysis engine and actionable optimisation suggestions to maximise ATS pass-through rates.

### AI-Powered Interview Preparation

Generates role-specific interview questions, evaluates answers, and supports live code execution for technical roles. Tracks session history with analytics and reporting.

### Communication Tools

- **Cold Email Generator** — Personalised outreach emails tailored to the job and the candidate's background
- **Cover Letter Creator** — Context-aware cover letters generated in seconds
- **LinkedIn Post Generator** — Professional posts to boost visibility during a job search

### User Management & Authentication

Secure authentication with personalised dashboards for managing resume history, interview sessions, and generated content.

## Prototype Overview

This repository contains the **frontend prototype** of TalentSync, built with a sleek **dark mode interface**. The UI is structured around a modular component library featuring:

- Responsive dashboard layouts
- Resume upload and analysis views
- Interview preparation interface with live code execution
- Communication tools panel
- ATS scoring and optimisation display

The dark mode design was intentionally chosen for a focused, professional feel that reduces eye strain during long job-search sessions.

## Why TalentSync Stands Out

- **All-in-one platform** — Resume analysis, ATS optimisation, interview prep, and outreach tools in a single product, not scattered across 5 tabs
- **Understands narratives** — Goes beyond keyword matching to extract real career impact and achievements
- **Dual-sided value** — Helps both job seekers sharpen their positioning and recruiters surface signal faster
